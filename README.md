# Hostel Complaint Management System

## 📌 Project Description
This project is a web application developed to manage hostel complaints efficiently. Students can raise complaints, track their status, and administrators can manage and resolve them.

---

## 🎯 Objectives
- To simplify complaint registration process
- To provide real-time status tracking
- To improve communication between students and hostel management

---

## 🚀 Features
- User Login System
- Raise Complaint
- View Complaints
- Track Complaint Status (Pending / In Progress / Resolved)
- Admin Dashboard
- Filter & Manage Complaints

---
